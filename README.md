# ProspectAI

## Como abrir no Claude Code

1. Baixe e descompacte o ZIP
2. Abra o Claude Code (claude.ai/download)
3. Digite: `cd` e arraste a pasta `prospectai-project` para o terminal
4. Cole essa mensagem para o Claude continuar:

---

"Leia o arquivo CONTEXT.md e continue o projeto ProspectAI. Próxima etapa: adicionar multiusuário com Supabase + Vercel."

---

## Arquivos

- `prospectai.html` — CRM completo funcionando no browser
- `CONTEXT.md` — contexto completo do projeto para o Claude Code
